# Personal Portfolio Website

A clean, minimal, Google-inspired portfolio website built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

## 🚀 Live Demo

[View Live →](https://your-username.github.io/portfolio)

## ✨ Features

- Fully responsive (mobile, tablet, desktop)
- Smooth scroll reveal animations
- Parallax blob effects on hero
- 3D card tilt on hover
- Sticky navbar with scroll effect
- Mobile hamburger menu
- Clean, minimal aesthetic inspired by Google's Material Design

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (Intersection Observer, pointer events)
- Google Fonts (DM Serif Display + DM Sans)

## 📁 Project Structure

```
portfolio/
├── index.html    # Main HTML structure
├── style.css     # All styles
├── script.js     # Interactions & animations
└── README.md     # This file
```

## 🔧 How to Customize

1. **Your name** — search for `Your Name` in `index.html` and replace
2. **Your photo** — replace the `.about-img-placeholder` div with an `<img>` tag
3. **Projects** — edit the three project cards in the `#projects` section
4. **Skills** — update the tags in the `#skills` section
5. **Links** — update email, GitHub, LinkedIn in `#contact`
6. **Resume** — add your PDF and update the download link

## 🚀 Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Source to `main` branch, `/ (root)` folder
4. Your site will be live at `https://your-username.github.io/repo-name`
